## 2. Requisitos Funcionais (RF)
* **RF01:** O sistema deve permitir o cadastro de produtos com descrição, preçode venda, unidade de medida e fabricante.
* **RF02:** O sistema deve realizar buscas de produtos por nome, código de barras ou fabricante.
* **RF03:** O sistema deve verificar a disponibilidade de estoque antes de confirmaruma venda.
* **RF04:** O sistema deve permitir o cadástro rápido de novos clientes durante ao atendimento.
* **RF05:** O sistema deve registrar vendas a prazo vinculadas ao cadastro do cliente em "Contas a receber".
* **RF06:** O sistema deve emitir comprovante detalhado ao emitir qualquer operação de venda.
* **RF07:** O sistema deve permitir que o gerente cadastre novos produtos e ajuste níveis de estoque manual.
* **RF08:** O sistema deve gerar relatórios de produtos mais vendidos e hitórico de compras por fornecdor.
